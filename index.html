Resumen de Desempeño Formativo

Biblioteca de Conceptos de Ingeniería

La base común es la FR-4 (Fibra de Vidrio) o la Fenólica (Baquelita). El grosor estándar de la capa de cobre es de 35 micras (1 oz/ft²). ¡Recuerda limpiar muy bien la superficie antes de transferir el diseño!

  • Ángulos: Evita giros de 90°. Usa ángulos de 45° para prevenir el "efecto antena" y facilitar el grabado químico.
  • Ancho de Pista: A mayor corriente, mayor ancho. Para señales pequeñas usa 10-15 mil; para potencia, mínimo 25-40 mil.
  • Aislamiento (Clearance): Espacio mínimo entre pistas para evitar arcos eléctricos.

Usamos Cloruro Férrico o Persulfato de Amonio. El líquido "ataca" el cobre expuesto y respeta el cobre protegido por el tóner o la tinta indeleble.

Tip Pro: Calienta ligeramente el ácido para acelerar la reacción (máx. 45°C) para evitar que se despegue la máscara protectora de tinta.
